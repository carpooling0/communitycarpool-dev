Font Stack

Type System Updated

Three typefaces define the Community Carpool voice. Montserrat brings impact and authority to headings. Playfair Display Italic provides a contrasting editorial accent. Inter handles all body copy with exceptional screen legibility. This pairing was established from the DAMAC corporate flyer, April 2026.

What changed (April 2026): Roboto (headings) and the generic system stack (body) are replaced. Montserrat Black (900) is now the display/hero font. Playfair Display Italic is a new accent for taglines and one-word emphasis. Inter is the official body font.
Primary Typeface — Headings & Display

Montserrat Updated

Replaces Roboto. Used for all headings (H1–H4), hero headlines, buttons, labels, and navigation. The Black (900) weight is the signature display weight — use it for maximum impact on hero sections and marketing materials.

Sizing Reference

Type Scale

Consistent sizing hierarchy. All headings use Montserrat; body/UI uses Inter.

Role Font Size Weight Line Height Usage
Display Montserrat 56–72px 900 (Black) 1.05 Hero headlines, print taglines
H1 Montserrat 40px 900 (Black) 1.15 Page titles
H2 Montserrat 32px 700 (Bold) 1.3 Section headings
H3 Montserrat 24px 700 (Bold) 1.4 Sub-section headings
H4 Montserrat 18px 600 (SemiBold) 1.5 Card titles, small headings
Tagline Accent Playfair Display Italic 20–32px 400i (Italic) 1.3 One-word accents, taglines
Body Inter 16px 400 (Regular) 1.75 Paragraph copy, descriptions
Label / UI Inter 14px 500 (Medium) 1.5 Form labels, button text, nav
Caption Inter 12px 500 (Medium) 1.4 Helper text, timestamps, badges
Code Monaco / Courier New 13px 400 (Regular) 1.6 Code, tokens, API values
Implementation

Loading the Fonts

Copy this snippet into the <head> of any page that uses the brand typefaces.

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Montserrat:ital,wght@0,400;0,600;0,700;0,900;1,400&family=Playfair+Display:ital@1&family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
CSS variables to use:
font-family: 'Montserrat', sans-serif; — headings, buttons
font-family: 'Playfair Display', serif; font-style: italic; — accent italic
font-family: 'Inter', sans-serif; — body, UI, labels
font-family: 'Monaco', 'Courier New', monospace; — code, technical
