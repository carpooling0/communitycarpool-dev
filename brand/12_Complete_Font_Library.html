System Stack (Fallback Fonts)

If custom fonts fail to load, these system fonts provide excellent fallbacks.

Primary Stack

font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', 'Oxygen', 'Ubuntu', 'Cantarell', 'Fira Sans', 'Droid Sans', 'Helvetica Neue', sans-serif;

Benefits: Uses native system fonts for optimal performance. Looks native on each platform (iOS, Android, Windows, Mac).

Font Performance Tips

Best Practices

  • Limit fonts to 2-3 typefaces — One serif/sans-serif for body, one for headings, one monospace for code
  • Use font-display: swap — Shows system font immediately, swaps when custom loads
  • Limit weights — Only load 400, 600, 700 weights unless specific need
  • Preload critical fonts — Improve perceived performance
  • Use Google Fonts — Excellent CDN performance, cached globally
  • Test at different sizes — Make sure fonts are legible at 12px, 14px, 16px, etc.
  • Consider accessibility — Avoid very thin weights (< 300) for small text
  • Match letter-spacing — Some fonts need tighter/looser spacing
