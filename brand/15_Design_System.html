Accessibility Guidelines

Keyboard Navigation

Action Key Behavior
Tab Through Tab Move focus to next interactive element
Reverse Tab Shift + Tab Move focus to previous interactive element
Activate Button Enter / Space Click the focused button
Toggle Checkbox Space Toggle checkbox state
Open Select Space / Arrow Down Open dropdown menu

Color Contrast

Element Foreground Background Ratio WCAG
Body Text #1F2937 #FFFFFF 13.6:1 ✓ AAA
Button Text #FFFFFF #10B981 5.8:1 ✓ AA
Secondary Text #6B7280 #FFFFFF 7.3:1 ✓ AAA

Best Practices

✓ Always provide: Meaningful labels for form inputs, Alt text for images, ARIA labels for icons, Sufficient color contrast (4.5:1 minimum)

Responsive Design

Breakpoints

Breakpoint Width Device Layout
Mobile < 640px Phone Single column, full width
Tablet 640px - 1024px Tablet 2-3 columns
Desktop > 1024px Desktop 3+ columns, full features

Responsive Utilities

/* Mobile First */ .component { display: block; } /* Tablet and up */ @media (min-width: 640px) { .component { display: grid; grid-template-columns: 2fr 1fr; } } /* Desktop and up */ @media (min-width: 1024px) { .component { grid-template-columns: 3fr 1fr 1fr; } }

Implementation Guide

CSS Class Structure

/* Buttons */ .btn { /* base styles */ } .btn-primary { /* primary variant */ } .btn-secondary { /* secondary variant */ } .btn-small { /* size variant */ } .btn-large { /* size variant */ } .btn-disabled { /* disabled state */ } /* Forms */ input, select, textarea { /* base styles */ } input:focus { /* focus state */ } input:disabled { /* disabled state */ } input.error { /* error state */ } .form-group { /* wrapper */ } .helper-text { /* helper text */ }
