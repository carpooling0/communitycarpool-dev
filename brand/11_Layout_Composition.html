White Space Guidelines

✓ Do:

  • Use consistent spacing from the spacing scale
  • Add breathing room around headings (24px below)
  • Separate sections with 32px vertical space
  • Use white space to guide visual hierarchy

✗ Don't:

  • Crowd content with no breathing room
  • Use arbitrary spacing values
  • Mix inconsistent gaps between components
  • Ignore white space for visual balance
