Extended Palette New

Two colors introduced from the DAMAC corporate flyer template. Use Warm Cream for print collateral, landing pages, and premium-feel backgrounds. Near-Black replaces cool-toned dark text on light or cream backgrounds.

#F8F4E6
Warm Cream New
RGB 248, 244, 230
Premium background — print flyers, landing page hero, collateral. Warmer and more considered than pure white.
--warm-cream
#1A1A1A
Near-Black New
RGB 26, 26, 26
Hero headline text on cream or light backgrounds. Warmer than the blue-tinted #1F2937.
--near-black
Usage rule for the new colors: Warm Cream (#F8F4E6) is for print and premium digital contexts — not the everyday web UI (which stays on #FFFFFF / #F9FAFB). Near-Black (#1A1A1A) is specifically for large display text (H1, hero taglines) on cream backgrounds. For body text on white, continue using Text Primary (#1F2937).

Implementation

Color Usage Reference

Where each color should and should not be used.

Color Use for Do not use for
Dark Forest Green Headings, footer, nav, logo, large text blocks on white Small body text (contrast too heavy), button fill
Bright Chartreuse Logo accents, graphic dividers, data highlights, illustrations Body text, small text (fails contrast on white)
Button Green Primary CTA buttons, links, active states, success indicator Large background fills, decorative only
Warm Cream New Print flyers, landing page hero sections, premium collateral backgrounds Web UI backgrounds (use #F9FAFB instead), dark text backgrounds
Near-Black New Hero display text (H1, taglines) on cream or white backgrounds Body copy (use #1F2937), captions
Text Primary All body copy, form labels, card text on white/off-white Headings on colored backgrounds
