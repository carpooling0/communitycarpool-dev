Spacing System

Consistent spacing scale for padding, margins, and gaps throughout the platform.

Spacing Scale

Token Value Usage
--xs 4px Minimal gaps, inline spacing, icon margins
--sm 8px Button padding, tight spacing, form gaps
--md 12px Standard button padding, component spacing
--lg 16px Card padding, section padding, default gaps
--xl 24px Between sections, list gaps, large spacing
--2xl 32px Major section breaks, header margins
--3xl 48px Page margins, hero section spacing

Shadow System

Elevation shadows for depth and hierarchy.

Token Value Usage
--shadow-sm 0 2px 8px rgba(0,0,0,0.06) Cards, subtle elevation
--shadow-md 0 4px 16px rgba(0,0,0,0.12) Modals, dropdowns, medium elevation
--shadow-lg 0 8px 32px rgba(0,0,0,0.15) Overlays, popovers, maximum elevation

Border Radius System

Consistent corner rounding for different component types.

Token Value Usage
--radius-sm 4px Small buttons, compact inputs
--radius-md 6px Standard buttons, cards, inputs
--radius-lg 8px Modals, panels, large components
--radius-full 50% Avatars, badges, circular elements

Visual Examples

4px • 6px • 8px • 50%

CSS Variables (Copy & Paste)

:root { /* Spacing */ --xs: 4px; --sm: 8px; --md: 12px; --lg: 16px; --xl: 24px; --2xl: 32px; --3xl: 48px; /* Shadows */ --shadow-sm: 0 2px 8px rgba(0,0,0,0.06); --shadow-md: 0 4px 16px rgba(0,0,0,0.12); --shadow-lg: 0 8px 32px rgba(0,0,0,0.15); /* Border Radius */ --radius-sm: 4px; --radius-md: 6px; --radius-lg: 8px; --radius-full: 50%; }

Usage Examples

Button with Tokens

.button { padding: var(--md) var(--lg); border-radius: var(--radius-md); box-shadow: var(--shadow-sm); }

Card with Tokens

.card { padding: var(--lg); border-radius: var(--radius-lg); box-shadow: var(--shadow-md); margin-bottom: var(--xl); }

Section Spacing

section { margin: var(--2xl) 0; padding: var(--3xl) var(--lg); }
