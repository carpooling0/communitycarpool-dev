    Asset Usage Guidelines

    Logo Usage

    Clear Space: Maintain minimum 10% of logo height as clear space

    Minimum Size: Never use logo smaller than 100px

    Colors: Use full-color on light backgrounds, monochrome on complex backgrounds

    Color Usage

    Primary: Use dark green for text, headings, navigation

    Accent: Use bright green for highlights and emphasis

    CTA: Use button green for all interactive elements

    Typography Usage

    Headings: Always use Montserrat 700+ weight

    Body: Always use Inter 400 weight for readability

    Code: Always use Source Code Pro for technical content

    File Format Recommendations

    SVG: Web, responsive, scalable (preferred)

    PNG: App icons, social media (24-bit transparent)

    PDF: Print, documents (CMYK conversion)
