Implementation Guide

Google Fonts Link

<link href="https://fonts.googleapis.com/css2?family=Inter:wght@100;300;400;500;600;700;800;900&family=Montserrat:wght@100;300;400;500;600;700;800;900&family=Source+Code+Pro:wght@200;300;400;500;600;700;900&display=swap" rel="stylesheet">

CSS Variables

:root { --font-sans: Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif; --font-serif: Merriweather, 'Georgia', serif; --font-mono: 'Source Code Pro', 'Courier New', monospace; --font-display: Montserrat, sans-serif; }

Usage Example

body { font-family: var(--font-sans); } h1, h2, h3 { font-family: var(--font-display); } code { font-family: var(--font-mono); }
