Google Maps & Mapbox

Google Maps Places API powers address autocomplete on the journey submission form. Mapbox is documented as a future option for displaying map visualisations. This page covers API key management, billing restrictions, and what each service does.

How autocomplete works

The journey submission form (index.html) loads the Maps JavaScript API with your API key. It initialises a google.maps.places.Autocomplete widget on the "from" and "to" address input fields. When a user selects a suggestion, the widget returns:

  • The human-readable formatted address (stored in submissions.from_location / to_location)
  • Latitude and longitude from the geometry (stored as PostGIS geography in from_point / to_point)

The coordinates are used by the find-matches edge function for spatial proximity matching.

API key security

Because the key is embedded in the frontend HTML, it is technically visible to anyone who views the page source. This is normal and accepted for Maps API keys — Google's protection mechanism is HTTP referrer restriction:

⚠️
The API key MUST be restricted by HTTP referrer. If the key is unrestricted, anyone who finds it can use it and generate charges on your account. Set the restriction immediately in Google Cloud Console if not already done.

Setting the HTTP referrer restriction

  1. Go to console.cloud.google.com
  2. Navigate to APIs & Services > Credentials
  3. Click on your API key
  4. Under Application restrictions, select HTTP referrers (websites)
  5. Add the following allowed referrers:
    • *.communitycarpool.org/*
    • localhost/* (for local development)
    • carpooling0.github.io/* (for the dev GitHub Pages site)
  6. Under API restrictions, select Restrict key and choose Maps JavaScript API and Places API
  7. Save

Monitoring usage and cost

In Google Cloud Console, go to APIs & Services > Dashboard to see request counts per API. Go to Billing > Reports to see cost breakdowns. The Places API charges per session or per request after the $200/month free credit is exhausted.

To set a budget alert:

  1. Go to Billing > Budgets & Alerts
  2. Create a budget for the project
  3. Set alert thresholds (e.g. 50%, 90%, 100% of monthly budget)
  4. Add email alert recipients

Mapbox (future)

Mapbox is not currently integrated. It is documented here as a planned future option for displaying interactive maps on the matches page (e.g. showing a shared route visually). Mapbox offers a generous free tier for map tile requests and has good support for custom styling.

If Mapbox is integrated in future:

  • The access token should be restricted by URL to communitycarpool.org
  • Store the token in frontend config (config.js), not in edge function secrets
  • The free tier supports 50,000 map loads/month
