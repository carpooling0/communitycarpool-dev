Admin Panel Guide (subset of these docs)
Admins and Support team members have their own focused guide covering only what they can see and do in the admin panel — no infrastructure, no secrets, no edge functions. It lives at docs/admin/ and is accessible from every team member's sidebar in the admin panel under Docs → Admin Panel Guide.

These Super-Admin Docs are a superset: everything in the Admin Panel Guide plus all infrastructure, database, deployment, and security content that only Super-Admins need.
