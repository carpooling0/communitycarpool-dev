Cost & billing

Community Carpool is designed to run at minimal cost. Most services are on free tiers or low-cost plans. This page documents what each service costs, where to manage billing, and how to set up spending alerts.

Supabase billing

Supabase pricing is based on:

  • Database size — free tier includes 500 MB
  • Edge function invocations — free tier includes 500,000/month
  • Bandwidth — free tier includes 5 GB/month
  • Realtime connections — not used by Community Carpool
  • Automatic backups — only available on Pro plan ($25/month+)

Check the current plan and usage at Supabase Dashboard > Settings > Billing. If nearing free tier limits, upgrade to Pro before usage spikes cause service interruptions.

ℹ️
Automatic backups require Pro plan. On the free tier, you must export backups manually via the SQL Editor or pg_dump. See Backup & recovery.

Resend billing

Resend's free tier supports:

  • 100 emails per day
  • 3,000 emails per month

For a growing user base, this may become a constraint. Monitor monthly email volume at resend.com/overview. When testing_mode is disabled and the user base grows, consider upgrading to a paid Resend plan. Login credentials: [REDACTED] — see password manager.

Google Maps billing

Google provides a monthly free credit ($200 as of 2024) for Maps Platform usage. The Places API (used for address autocomplete) charges per session or per request after the free credit is exhausted. Monitor usage in Google Cloud Console > Billing > Reports. Login: [REDACTED] — see password manager.

To minimise cost:

  • Restrict the API key to *.communitycarpool.org/* so it cannot be abused by third parties (see Google Maps & Mapbox)
  • Consider setting a budget alert in GCP Billing to notify when monthly spend exceeds a threshold

Setting up billing alerts

It is strongly recommended to configure spending alerts for each paid service:

  • Supabase: Settings > Billing — set a spend cap or upgrade notifications
  • Google Cloud: Billing > Budgets & Alerts — create a budget with email alerts at 50%, 90%, 100% of threshold
  • Resend: Check the dashboard for usage warnings; consider setting a daily send limit as a safety net
💡
Review all service dashboards at the start of each month to track usage trends. This takes about 10 minutes and can prevent surprise bills.
